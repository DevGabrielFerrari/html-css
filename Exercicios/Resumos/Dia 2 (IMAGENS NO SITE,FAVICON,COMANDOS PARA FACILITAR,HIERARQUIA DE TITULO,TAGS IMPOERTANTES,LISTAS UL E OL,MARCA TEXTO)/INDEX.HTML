Imagens no site

  1. Tag para Imagem = < img >
  2. Na opção src use o nome e local da pasta(se tiver) Ex: src="ctrl espaço e local da pasta"
  3. Coloque o nome da imagem na parte de alt = 'nome da imagem (você escolhe)'
  4. A imagem aparecerá GIGANTE para redimensionar vá no GIMP clique em (imagem,redimensionar,reduza,saia,arquivo,exportar como,salva)
  5. Se quiser utilizar imagens da web é só copiar o endereço e colar em src

Favicon (logo na aba do site)

Como deixar a imagem compativel?

  1. Selecione sua imagem
  2. Vá até o site Favicon.io
  3. Coloque a opção no caso png -> ico
  4. Baixe a imagem extraia na pasta do exercicio

Qual é a tag?

  1. Vá em cima da opção title que fica na head
  2. digite "link":favicon"
  3. Você só vai mecher na parte href=""
  4. Coloque o local da imagem usando ctrl espaço

Hierarquia de titulos

  • h1 = Titulo Principal (dentro do site)
  • h2 = Subtitulo
  • h3 = Subtitulo menor
  • h4 = Subtitulo menor ainda
  • h5 = Subtitulo de nivel minimo
  • h6 = Subtitulo mínimo
