Links e âncoras

  • Use a Tag A para colocar links em palavras
  • Na parte href="coloque o link aqui"
  • (Opcional) se você quiser que o link abra em outra aba e não sobresaia no seu site NA FRENTE do link Use target="_blank" rel="external"

Links internos

  • rel = "prev" = voltar uma página
  • rel = "next" = avançar

    Isso não muda nada é só significado

  • Para links internos (suas páginas do mesmo site) no href="pag.002.HTML"

    rel="nofollow" = Pedir para o navegador não dar bola (usado em anuncios)

    Qual é o código para adptar?

    1. Tag picture
    2. Digite dentro da tag source media e aperte Enter
    3. Troque min-widht por max-widht
    4. Na parte max-widht coloque o número de pixels (tamanho) com 50 px a mais
      5. Exemplo: source media = "(max-widht:750 px)" srcset ="imagens/fotosp.png" type="imagem/png">
    5. Dependendo do zoom do navegador a imagem pequena não seja para resolver ctrl shift -
