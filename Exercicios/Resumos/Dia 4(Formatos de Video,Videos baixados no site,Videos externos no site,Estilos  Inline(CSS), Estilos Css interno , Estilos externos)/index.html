Formatos de Video para seu site

  • Formatos: MP4,WebM,OGG

Como comprimir?

  • Vá no Hand Brake
  • Selecione o vídeo
  • Use o formato creator 760x60

Videos hospedadas no site (imagens baixadas)

  • Use a tag vídeo Ex: < video controls widht = "200">
  • < source src = "arquivo aqui" type = "video/mp4"

Vídeos Externos (ex:videos do youtube)

  • Entra no vídeo que você quer colocar
  • Vá em compartilhar e incorporação copia e COLA

Estilos Inline

  • Use somente em casos extremamente especificos vá na tag ex: <h1 style = "background-color; COR; font-family:Arial,Hebvetica,sans-serif">

Estilos CSS internos (local do arquivo principal)

  • Isso é só se for só um documento só se for 2 ou mais use o outro método
  • Debaixo do title na parte head
  • Coloque o NOME da tag que você quer estilizar e na frente use {}. Ex: 
    body{
        }

Estilos CSS externos (Método mais eficiente)

  1. Abra um arquivo style.css
  2. Nos arquivos que você quiser usar vá debaixo do title e use a tag link:css
  3. no href="style.css"
  • Você pode usar um método dentro do outro sem problema
  • Você pode usar quantos arquivos de style você quiser

Tags do Dia

  • "background-color: sua cor;" = cor de fundo da página

  • "font-family: Arial,hebervetica,sans-serif;" = Muda a fonte do body POR EXEMPLO

  • "font-size: 20px;" = "zoom" da pagina

  • "color: darkred; = cor da tag" = Ex: Se for o h1 da página irá modificar a cor todos os titulos

  • "text-align: justify;" = organizar texto (separar mais as palavras)
