Representando Cores com CSS

    Representação por HSL

  • hsl(hue,sturation,luminosity)
  • Ex: hsl(240,100%,50%)

    • Representação por código Hexadecimal

  • Ex: #ffffff = white

    • Método mais Útil

  • Representação RGB (Red,Green,Blue)
  • Você pode escolher a cor e clicar no botão direito e arrastar até chegar na sua cor
  • Você pode ir no site coolors.co,Adobe color,palettons e copiar e colar o código da cor

Harmonia de Cores

  • Cores Primárias = Amarelo,Azul,Vermelho
  • Cores Segundarias = Laranja,Violeta,Verde
  • Cores Terciarias = Amarelo esverdeado,Amarelo alaranjado, Vermelho alaranjado, azul arroxeado

  • Cores Frias = Todos os verdes,Azul,Violeta.
  • Cores quentes = Roxo,amarelo alaranjado,laranja,vermelho

  • Paleta de 3 a 5 cores no MÁXIMO!

  • No site do cliente use a cor primária da logo dele (A cor que mais se destaca)

  • Contarste Grande = Use a cor oposta no circulo cromático
  • Contarste Médio = Pegue as as duas primeiras cores vizinhas no circulo cromático.
  • Contraste Médio relacionado = Escolha a sua cor principal e uma cor vizinha pule uma e pegue a cor pule uma e pegue a cor (pra direção da cor vizinha no circulo)
  • Cores intercaladas = Escolha sua cor principal Pule uma e Pegue a outra e assim vai Ex: Amarelo,Laranja,Vermelho (Mas a cor fica rigida(dura))

  • Cores triadicas = Pega uma pula três pega a outra pu...
  • Cores em Quadrado = Pega uma pula duas pega uma pu...
  • Cores Tetradicas = Pega duas cores que você quiser e escolha o oposto delas formando tipo um X sla

  • Monocromia = Mexer na luminosidade,contraste na cor causando efeito de degrade.

Como criar degrade em CSS

  1. Debaixo do title use a tag style
  2. Dentro da tag use body{}
  3. Dentro da tag body coloque background-image:linear-gradient(to right,white,blue)

  4. to right = pra direita
  5. to left = pra esquerda
  6. to top = pra cima
  7. to bottom = pra baixo (quase não funciona)

  8. O "white,Blue" = De branco para Azul

Criando um exemplo Real

  1. Crie um link para estilo CSS debaixo do title e vamos lá

    • Use a regra @charset "UTF-8"; para não dar problema de acentuações

  2. main = diz que aquele é o conteudo principal

  3. Configuração Global do site: *{}

  4. Escolha um paleta
  5. No main que é o seletor daquele texto dentro de 
    
main{
    background-color: white (Vai formar um quadrado branco atrás do texto)
    width: 600px (Largura)
    padding: 10px (afastar a frase da borda do fundo do texto)
    margin:auto; (centraliza a tela no meio)
}
